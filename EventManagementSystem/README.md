# Event Management System

## Project Description
The Event Management System is a Java-based web application developed using Servlets, JSP, and JDBC.  
It allows an admin to manage events and demonstrates core concepts of Java Web Development such as MVC architecture, database connectivity, form handling, and session management.

## How to Run the Project
1. Import the project into Eclipse or any Java IDE  
2. Configure Apache Tomcat Server  
3. Run the `database.sql` file in MySQL  
4. Update database username and password in `DBConnection.java`  
5. Deploy and run the project on the server
